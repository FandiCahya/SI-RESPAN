Tentang Aplikasi

Aplikasi Penentuan Responden Survei Harga Pangan dirancang untuk menyederhanakan proses pengumpulan dan analisis data harga pangan strategis.

  • Import Data Fleksibel
    Memungkinkan admin mengunggah data harga dari file .xlsx dengan format yang dinamis.
  • Analisis Otomatis
    Menjalankan K-Means Clustering dan SRS untuk menentukan responden, serta menghitung statistik penting.
  • Visualisasi Peta
    Menampilkan lokasi responden terpilih di peta interaktif dengan fitur filter per komoditas.
  • Manajemen Hak Akses
    Membedakan fitur yang dapat diakses oleh Admin dan Enumerator untuk keamanan data.
